# Hovac-Texture-Pack
A custom Minecraft Bedrock Edition (MCBE) texture pack designed for a unique gameplay experience.

## 📁 Repository Structure
- **src/**: Contains the core pack files (manifest, icon, and textures).
- **particles/**: Custom particle effects.

## 🚀 Installation
1. Use the **Texture pack installer.bat**
2. Activate the pack in **Settings > Global Resources**.

## ✨ Features
* [ ] High-fidelity textures
* [ ] Custom UI elements
* [ ] Optimized for performance

## 🛠️ Development
If you want to contribute or modify the pack:
1. Clone the repo: `git clone https://github.com/P4pawan/Hovac-Texture-Pack.git`
2. Make your changes in the `/src` folder.
3. Test in-game using the "Development Resource Packs" folder.
